# 线程池

要想批量创建线程，同时给与每个线程分配任务，就需要线程池。

主要功能：

  1. 提前性，在分配任务前，就提前创建好线程，供给分配。
  2. 复用性，结束任务的线程放回线程池，供给下一次使用。

要满足创造线程的数量恒定，以及线程结束后的复用，那么就需要用到队列。
