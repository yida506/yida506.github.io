# Android 组成部分
  • Activity 表示层 展示 UI，响应用户动作
  • Service 更新数据源和 Activity
  • Content Provider 数据储存器
  • Intent 消息传递框架
  • Broadcast Receiver Intet 监听器
  • Widget 可视化组件
  • Notification 向用户发起信号

Manifest 存储在最底层，定义应用程序的组件和需求的结构和元数据。

# Activity

该类为所有用户界面的基础。

每一个 Activity 都表示一个屏幕，类似 web 中的 html
