# 关联挖掘

# 基本概念

频繁项集：n 项的集合。

支持度：代表 A、B 同时出现的概率，如果概率小，那么说明 A 与 B 的关系不大，如果 A、B 总是同时出现，那么说明他们是相关的。

support=P(AUB)support = P(AUB)support=P(AUB)

置信度：包含 A 的数据集中包含 B 的百分比。

# Apriori 算法

使用候选项集找频繁项集

核心思想，只要一个项集是非频繁的，那么包含他的项集就都是非频繁的。
