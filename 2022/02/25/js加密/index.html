# 算法加密
# MD5

本质上是一种信息摘要算法，对应关系为 n 对 1，也就是不同的数有相同的加密结果，但是一个数只会有唯一的加密值。密文一般是 16 或 32 位。

md5 加密特征：

  1. 0123456789ABCDEF、0123456789abcdef 的出现
  2. 1732584193、-271733879、-1732584194、271733878 的出现

至于为什么会有上述特征，后面在研究。

# RSA 加密
