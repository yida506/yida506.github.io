# Native 基础

一般来讲，在 Android 开发中，除了使用 Java 语言开发并编译为 dex 文件然后由 ART 虚拟机执行的方式外，和 web 端的 wasm 一样，移动端也纯在套用 c 代码进行开并编译执行的方式 (通过 java Native Interface 可以调用 C/C 函数)。

优点:

  • 性能
  • 安全
