# 反调试

所谓反调试就是在网站中嵌入 debugger，然后对控制台进行检测，若打开了控制台则进入无限 debugger;

解决方案:

  • 直接右键 never pause here
  • 若采用定时器的形式调用，将其置空

  • 若在原型链上

  • 若位 eval 调用，hook 一下即可
